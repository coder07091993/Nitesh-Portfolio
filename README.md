# Nitesh Kumar - Portfolio

Personal portfolio website showcasing 9+ years of Staff Software Engineering experience.

## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
cd nitesh-portfolio
vercel
```

### Option 2: GitHub + Vercel Dashboard
1. Push this folder to a GitHub repository
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repository
4. Framework preset: **Other**
5. Click **Deploy**

### Option 3: Drag & Drop
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag the entire `nitesh-portfolio` folder into the browser

## Local Preview
```bash
npx serve .
```
